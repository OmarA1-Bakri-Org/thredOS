# Search

Research and retrieve information from the web and local knowledge sources.

## Intent

Use this skill when a thread needs evidence, comparisons, or source-backed facts before implementing or deciding.

## When To Use It

- compare APIs, libraries, or product behavior
- verify a fact that may have changed
- gather citations for a launch or pricing decision
- inspect repo-local docs and combine them with external sources

## Safe Boundaries

- prefer primary sources
- separate facts from inference
- summarize findings concisely enough for the next agent to act on them

## thredOS Examples

- research a dependency before binding it to a canonical agent tool loadout
- gather evidence for a UX or pricing decision
- verify a product claim before it is added to launch copy
