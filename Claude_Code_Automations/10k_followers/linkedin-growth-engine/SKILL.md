# LinkedIn Growth Engine

A sequential system for organic LinkedIn growth using AI tools. Built on the
principle: write for one reader, never about yourself.

## System Flow

1. **Foundation** (do once) -- Mission, ICP, profile, headshot
2. **Content Engine** (daily) -- Research, design, write, format
3. **Engagement Loop** (daily) -- Warm-up, post, warm-down, outreach

## How to Guide Users

When a user triggers this skill, determine where they are in the system:

- **No mission/ICP yet** -- Start with Foundation (Section 1 in references/foundation.md)
- **Profile not optimised** -- Go to Profile Checklist (references/foundation.md)
- **Need content ideas** -- Go to Content Engine (references/content-engine.md)
- **Need post templates** -- Go to Templates (references/content-engine.md, Section 3)
- **Want to grow faster** -- Go to Engagement Loop (references/engagement.md)
- **Need AI tool recommendations** -- Go to Tool Stack (references/tools.md)
- **Want a full plan** -- Go to 90-Day Calendar (references/calendar.md)

## Core Rules (Apply to All Advice)

1. Every post is about the reader's problem, never the creator's achievements
2. Target ONE specific person (ICP), not "everyone"
3. Hook: exactly 2 sentences, ~55 characters each
4. Image: 1080x1350 (4:5), scrappy/unbranded, never looks like an ad
5. Daily consistency is non-negotiable -- "Never miss a day"
6. Warm up account with 10-15 comments BEFORE posting
7. No hashtags, no emojis in post copy
8. 80% of results come from winning concepts -- double down on what works
9. AI accelerates execution but requires human judgment and taste
10. 1 hour/day total: 20 min content + 30 min engagement + 10 min outreach

## Quick-Start (For Users Who Want Immediate Action)

If the user wants to start RIGHT NOW, give them this sequence:

1. Define mission in one sentence: "I help [WHO] do [WHAT] so they can [OUTCOME]"
2. Define ICP: Give their ideal reader a name, age, role, and one specific struggle
3. Write a hook: 2 sentences, ~55 chars each, pattern interrupt
4. Create a 1080x1350 image: search Pinterest for `[niche] + infographic`, remix in Gemini
5. Post it. Then comment on 10 other posts.
