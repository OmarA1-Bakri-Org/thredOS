## Recommended Stack (Minimum Viable)

For users who want to start with the smallest possible toolset:

1. **Mission-GPT** (free) -- define your mission
2. **ICP-GPT** (free) -- define your reader
3. **Google Gemini** (free) -- headshot + image generation
4. **Claude AI** (free) -- post writing
5. **Canva** (free) -- image editing
6. **Pinterest** (free) -- visual inspiration

Total cost: $0
