# 90-Day Implementation Calendar

## Week 1-2: Foundation

| Day | Task | Time | Deliverable |
|-----|------|------|-------------|
| 1 | Define mission using Mission-GPT | 30 min | One-sentence mission statement |
| 1 | Define ICP using ICP-GPT | 30 min | Detailed persona document |
| 2 | Generate AI headshot using Gemini | 20 min | 1080x1080 profile picture |
| 2 | Optimise profile (headline, banner, featured, about) | 30 min | Fully optimised profile |
| 3 | Set up tool accounts (Gemini, Canva, Gamma, EasyGen) | 30 min | All tools ready |
| 3 | Create Chrome folder with 15 niche creator activity feeds | 15 min | Bookmark folder |
| 4 | Use Color-Theory GPT for brand colours | 15 min | Brand colour palette |
| 4-5 | Create first 3 posts using content formula | 2 hrs | 3 ready-to-publish posts |
| 5-7 | Publish first 3 posts (1/day), begin warm-up/warm-down | 1 hr/day | First 3 posts live |
| 8-14 | Daily posting + engagement routine | 1 hr/day | 7 more posts, engagement habit |

## Week 3-4: Content Engine Live

| Day | Task | Time | Deliverable |
|-----|------|------|-------------|
| 15 | Use Calendar-GPT: generate 30-day calendar from best post | 30 min | 30 content ideas |
| 15-28 | Execute daily engine: Research, Design, Write, Post, Engage | 1 hr/day | 14 more posts |
| 21 | Create first carousel post using Gamma.app | 45 min | Carousel PDF |
| 28 | Review analytics: identify top 3 performing post types | 30 min | Analytics insights |

## Week 5-6: Double Down on Winners

| Focus | Actions |
|-------|---------|
| Format optimisation | Create 2 carousels per week. Increase single-image infographic posts. |
| Content refinement | Analyse which hooks perform best. Double down on winning angles. |
| Engagement depth | Start replying to comments with follow-up questions to extend threads. |

## Week 7-8: Expand Research

| Focus | Actions |
|-------|---------|
| Reddit mining | Use Red Searcher GPT and Grok weekly to mine Reddit communities. |
| Content backlog | Build a backlog of 20+ content ideas so you never face a blank page. |
| Visual variety | Experiment with handwritten-style infographics vs. clean data visualisations. |

## Week 9-10: Hook Mastery

| Focus | Actions |
|-------|---------|
| Hook A/B testing | Try all 5 hook techniques. Track which pattern interrupts work best. |
| Open loop refinement | Experiment with different open loop structures in hooks. |
| Performance tracking | Create a spreadsheet: hook type vs. impressions vs. followers gained. |

## Week 11-12: Community Building

| Focus | Actions |
|-------|---------|
| DM expansion | Increase outreach to 10 connection requests per day. |
| Relationship building | Start engaging with commenters' own posts. Build reciprocal relationships. |
| Collaboration | Identify 3-5 creators at similar follower count. Cross-promote or collaborate. |

## Milestone Expectations

| Week | Aggressive | Moderate |
|------|-----------|----------|
| 2 | 500-1,000 followers | 200-500 |
| 4 | 2,000-5,000 | 500-1,500 |
| 8 | 5,000-15,000 | 2,000-5,000 |
| 12 | 10,000-30,000 | 5,000-10,000 |

**Aggressive**: Daily posting + full warm-up/warm-down + 5 connection requests/day.
**Moderate**: 4-5 posts/week with partial engagement routine.

Maria Zhanette hit 10K in 17 days -- aggressive end of the spectrum.

## Daily Checklist (Post-Foundation)

Use this every day after Week 2:

- [ ] Research: 5 min with Grok/Perplexity for a pain point
- [ ] Design: 5 min Pinterest reference, 5 min Gemini remix, 2 min Canva touch-up
- [ ] Write: 3 min body in Claude/EasyGen, 2 min hook generation
- [ ] Warm-up: 15 min commenting on 10-15 posts
- [ ] Publish: 2 min
- [ ] Warm-down: 15 min commenting + replying to own post comments
- [ ] Outreach: 10 min DMs + 5 connection requests
